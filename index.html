About

I worked as a logistician by delivery services. And all the time I was looking for tools, that could help with routine work. Once upon a time I found it. It was software especially made for delivery service. Since then I described almost all business-processes for software engineer and the software became popular. At the moment many delivery companies are using it.

My first experience in the WEB-development I've got in 2011. There was the website for my own company. It was made with pure HTML & CSS. The second website I made with Joomla. It was really exiting, but I had others goals at that moment...

After I sold my company, I returned to work by other delivery services, but work there was no longer fun. After working for a few more years, I finally decided to change the field of activity.

I remembered that I was excited about web-development, learned the possibilities for starting with the profession. It turned out that this is possible to start at any age. In addition, this activity provides endless opportunities for professional growth, there is a huge number of free high-quality resources for self-education and a large community of developers. I like to see the result as soon as possible, that's why I chose the frontend-development.

I believe that my ability to learn and develop new skills will help me become an experienced Frontend developer.
